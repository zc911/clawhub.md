## 🏗️ 技术栈

### 前端
- React 18 + TypeScript
- Vite
- TanStack Query (数据获取)
- Tailwind CSS (重构样式)
- shadcn/ui (组件库)
- React Router v6
- Zod (表单验证)
- Lucide Icons

### 后端
- Node.js + TypeScript
- Express
- PostgreSQL + Prisma ORM
- Redis (缓存)
- BullMQ (队列)
- Passport.js (GitHub OAuth)
- Stripe SDK (支付)

### DevOps
- Docker + Docker Compose
- GitHub Actions (CI/CD)
- Prometheus + Grafana (监控)
- Sentry (错误追踪)
