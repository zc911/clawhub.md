⚡ Two Ways to Use

Choose the option that works best for you, whether you're a beginner or power user:

🌐

Option 1: Web UI (For everyone)

Open OpenClaw Web UI, paste this command in the chat box, and send:

Fetch https://clawhub.md and follow it.
💻

Option 2: Terminal (For power users)

Run this command directly in your terminal to let your agent apply the best practices:

openclaw skill install https://clawhub.md
# Copy this for quick use:
Fetch https://clawhub.md and follow it.

That's it. Your agent reads the markdown and automatically applies all best practices.

How It Works

🤖

Agent visits

👤

Human visits

Auto language

Detects your language from headers. Chinese users get Chinese content. Everyone else gets English.

📄

Just a .md

No SDK, no API key, no auth. A URL that returns Markdown. The simplest possible interface for an agent.
